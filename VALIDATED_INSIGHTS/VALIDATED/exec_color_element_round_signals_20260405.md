## How These Signals Arose

### The Paradox Resolved

**Original theory** (Exec/Upside): "Higher Exec scores should universally outperform."
- **Finding:** Correlation near-zero (r=0.004), no universal effect.
- **Why?** Exec is a **structural score** (BaZi math), not a direct performance predictor.

**New finding** (Exec × Color × Dimension): Exec DOES predict performance, but only when combined with:
- **Color** (the player's rhythm color, unchanging)
- **Element** (Wu Xing: Earth, Fire, Metal, Water, Wood — yearly)
- **Round Type** (Open, Positioning, Closing, Survival)

**Mechanism:** Color + Exec create a **synergy**. Some color-exec combos are harmonious (Green+Exec75+Wood), others dissonant (Orange+Exec25+Fire).

## Deployment Guide

### For 2-Ball Matchups

**When comparing two players:**

1. **Look up each player's Color** (from PLAYERS sheet, Column G)
2. **Check current Year's Element** (2026 = Fire)
3. **If Round Type is known** (Open, Closing, etc.)

**Apply signal:**
- **Green × Exec 75-100 in Wood year** = +0.876 edge (now Wood ended, check for 2026)
- **Red × Exec 50-75 × Closing** = +0.936 edge (if known to be closing)
- **Orange × Exec 50-75 × Open** = −0.654 edge (AVOID/FADE)
- **Orange × Exec 25-50 × Fire (2026)** = −0.571 edge (AVOID/FADE)

### Kelly Adjustment

For a baseline matchup (52% win rate):
- **Positive signal (+0.7 vs_avg):** Apply +2% Kelly multiplier (more aggressive)
- **Negative signal (−0.5 vs_avg):** Apply −2% Kelly multiplier (reduce exposure)

### Risk Considerations

1. **Exec buckets (0-25, 25-50, 50-75, 75-100)** are broad. Further sub-bucketing might reveal noise.
2. **Element changes yearly** (2026 = Fire). Previous year analyses won't apply.
3. **Negative signals are more robust** (larger samples, higher reliability). Use these with confidence.
4. **Positive signals with small test samples** (Orange×Exec75×Earth, n=40) should be weighted lower.
