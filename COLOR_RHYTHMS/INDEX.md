## 📚 What This Is

The Color Rhythms system is the foundational reference material for Luckify Me's color-based predictive model. It documents:
- **8 Color Identities** — the archetypal energies (Pink, Orange, Blue, Yellow, Green, Purple, Red, Brown)
- **Traits per Color** — the 6+ characteristics that define each color's behavior
- **Chakra Positions** — the energetic anatomy linked to colors
- **17 Windows** — score ranges and directives for timing interpretation
- **Wave Patterns** — momentum and timing cycles

## 🔄 How This Is Used

### In Golf Engine (Current)
- **Daily Color:** BaZi engine calculates which color applies to each player on each day
- **Golf Scoring:** Colors map to Exec/Upside/Peak modifiers
- **Reference:** This folder documents what each color means (traits, patterns, directives)

### Future (Birth Color Integration)
- **Birth Color:** Channeled once per player, never changes (1-4 colors per player)
- **Synergy:** Daily Color × Birth Color → enhanced predictions
- **Player Profiling:** Birth color traits become part of player models
