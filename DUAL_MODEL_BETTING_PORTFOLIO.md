## RISK MANAGEMENT FOR DUAL-MODEL PORTFOLIO

### Key Risks

**Risk 1: Model Correlation Drift**
- If both models suddenly agree on same combos (correlation >0.5), lose diversification benefit
- Mitigation: Monitor correlation weekly; diverge if >0.5

**Risk 2: Sequential Losses from Both Models**
- If both have bad week simultaneously, drawdown can be steep
- Mitigation: Cap total portfolio at 50% bankroll; use half-Kelly for Zodiac signals

**Risk 3: Overfitting to Dual-Model "Consensus"**
- Tempting to only bet combos both models validate, but these are currently weak
- Mitigation: Bet each model independently; use separate capital allocation

### Stop-Loss Rules

**For Element Signals (Tier 1):**
- If any signal drops below 50% win rate over 25 plays → suspend
- If combined Element portfolio drops >8% below expected edge → revalidate thresholds

**For Zodiac Signals (Tier 2):**
- If win rate drops below 48% over 30 plays → suspend
- If Zodiac model shows >10% variance in edge estimates → revalidate

**For Entire Portfolio:**
- If combined portfolio drops >15% from expected performance over 4 weeks → full audit
- If correlation between models exceeds 0.6 → shift allocation
