# BETTING INSIGHTS — Math × Intuition

**Purpose:** Bridge validated statistical signals with deeper archetypal understanding. Every betting insight includes both the evidence layer (math) and the intuition layer (Soltar domain knowledge).

**Structure:**

---

## 📚 How This Works

**Layer 1 (Math):** Statistical evidence
- p-values, sample sizes, effect sizes
- Out-of-sample validation
- Correlation to actual performance

**Layer 2 (Soltar):** Archetypal understanding
- Why the pattern exists (color theory, energy dynamics)
- What the market misses (overpricing luck, underweighting skill)
- Predictive stability (will this hold? yes if archetypal, no if statistical fluke)

**Layer 3 (Deployment):** Betting application
- Which signals are Kelly-appropriate
- When to size up, when to pass
- How market pricing changes our edge calculation
