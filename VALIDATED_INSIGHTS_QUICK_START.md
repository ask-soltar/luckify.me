# VALIDATED INSIGHTS — Quick Start Guide

**🎯 Ready to start? Read this first (5 minutes).**

## What This Is

A system for testing your intuitive theories against real golf data.

**The fusion:** Spiritual/intuitive knowledge + mathematical proof = validated insights

---

## How It Works (3 Steps)

### 1️⃣ You Share a Theory
```
"I think Purple players close stronger in Calm conditions"
```

### 2️⃣ Claude Tests It
- Designs statistical test
- Runs analysis on golf data (2022-2026)
- Checks 5 validation gates

### 3️⃣ Theory Gets Rated
- **🟢 VALIDATED** — Ready for use in signals
- **🟠 PARTIALLY_BACKED** — Interesting but needs more data
- **🔴 REJECTED** — Doesn't hold up

---

## The 5 Validation Gates

Every theory must pass all 5 gates to be **VALIDATED**:

1. **Statistical Significance** — Real effect, not random (p < threshold)
2. **Adequate Sample Size** — Enough data (n ≥ minimum)
3. **Meaningful Effect** — Practically important (Cohen's d or win rate diff)
4. **Stable Across Contexts** — Works in multiple conditions (time, subgroup, etc.)
5. **Not Luck** — Repeats in test data (not just training data)

**Thresholds are flexible by context.** A quick discovery test has looser gates than a betting signal.
